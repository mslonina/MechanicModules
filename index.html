The Mechanic Modules

Sample numerical modules for the Mechanic:
  • Hello — not so basic usage of the Mechanic
  • Mandelbrot — computes the Mandelbrot set
  • ArnoldWeb — computes the web of resonances of the Hamiltonian system described in Froeschle et al (Science, 2000)

Dependencies

  • GCC 4.6+ or equivalent
  • HDF5 1.8+, OpenMPI 1.4.3+
  • CMake 2.8+

Installation

$ cd {MODULE}
$ mkdir build
$ cd build
$ CC=mpicc cmake ..
$ make
$ sudo make install
